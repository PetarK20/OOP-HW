## Компилация и Изпълнение

За да компилирате, трябва да укажете на компилатора (`g++`) да добави директорията `include/` към пътя за търсене на заглавни файлове (чрез флага `-Iinclude`).

1.  Отворете терминал в основната директория на проекта (`OOP-HW/`).
2.  Изпълнете командата за компилация, която включва всички `.cpp` файлове и посочва include пътя:

```bash
g++ -std=c++17 -Wall -Wextra -O2 -Iinclude main/Test.cpp src/*.cpp -o library_app
